# Security Policy

MetaFinder is a small, feature-complete project. The supported version is the current `main` branch.

Please do not upload sensitive images to deployments you do not control. Image metadata can contain private information, including GPS coordinates, device details, timestamps, and author fields.

If you find a vulnerability, please report it through GitHub Security Advisories when available. If that is not available, open an issue with a minimal description and avoid posting sensitive files, secrets, or personal metadata.
